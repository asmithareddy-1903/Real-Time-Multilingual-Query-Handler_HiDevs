# Multilingual_Support

This project translates customer queries from multiple languages into English using an AI-powered language detection and translation pipeline. It provides a simple and efficient way to handle global support requests across different languages.

---

## Features

- Detects the language of incoming text  
- Translates any supported language to English  
- Displays the detected language and translated output  
- Lightweight & easy to integrate  
- Ideal for global customer support teams  

---

## How It Works

1. User enters a message in any language  
2. The system automatically detects the language  
3. The message is translated into English  
4. The output is displayed in a clean and readable format  

---

## Example

**Input:**  
Hola, ¿cómo puedo cambiar mi contraseña?

**Output:**  
- **Language:** Spanish  
- **Translation:** How can I change my password?
